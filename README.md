# Regional Sales Performance

Sales data analysis with Python EDA and an interactive Power BI dashboard showing regional performance trends.

## What it does
Analyses regional sales data to identify top performing regions, revenue trends, and sales patterns. Includes a Power BI dashboard for interactive exploration.

## Tech stack
| Area | Tools |
|------|-------|
| Analysis | Python, Pandas |
| Visualisation | Power BI, Matplotlib, Seaborn |
| Notebook | Jupyter |

## Files
- `regional_sales_analysis.ipynb` — full exploratory data analysis
- `dashboard.pbix` — interactive Power BI dashboard
- `dashboard.pdf` — static export of dashboard

## Skills demonstrated
- Exploratory data analysis with Python
- Data visualisation with Power BI
- Pandas data manipulation
- Business insights from sales data
